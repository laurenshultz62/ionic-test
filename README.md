# ionic-test

Clone
`git clone https://github.com/laurenshultz62/ionic-test.git`

`cd ionic-test`

Install Ionic
`npm install -g @ionic/cli native-run cordova-res`

Install Packages
`yarn install`

Start Ionic Web
`ionic serve`

Start Ionic iOS
`ionic cap run ios -l --external`

Start Ionic
`ionic cap run android -l --external`
